# ТикиТок

Учебный проект по созданию видеохостинга на Django

## Функционал
- Регистрация, вход и выход пользователей
- Разграничение ролей (гость, пользователь, администратор)
- Загрузка видеофайлов
- Лайки и дизлайки
- Комментарии к видео
- Админ-панель с возможностью блокировки видео
- 
## Стилистика платформы
- Тип: Lifestyle / Минимализм
- Аудитория: Молодёжь, студенты
- Tone of Voice: Дружеский, неформальный
- Цветовая гамма: Тёмная тема с розовыми акцентами (#ec4899)
- Единый стиль: Все страницы в одной цветовой схеме
- Превью: JPG/PNG/GIF до 2 МБ, 16:9
