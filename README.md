# ТикиТок — видеохостинг на Django

Учебный проект по созданию видеохостинга.

## Функционал

- Регистрация, вход и выход пользователей
- Разграничение ролей: гость, пользователь, администратор
- Загрузка видеофайлов с превью
- Лайки и дизлайки (сердечко и разбитое сердечко)
- Комментарии к видео
- Админ-панель с возможностью блокировки видео
- Кастомная тёмная тема с розовыми акцентами

## Стилистика платформы

- **Тип:** Lifestyle / Минимализм
- **Аудитория:** Молодёжь, студенты
- **Tone of Voice:** Дружеский, неформальный
- **Цветовая гамма:** Тёмная тема (#08080b) с розовыми акцентами (#ec4899, #be185d)
- **Адаптивность:** Корректное отображение на мобильных устройствах и десктопе

## Технические требования к превью

- Размер: до 2 МБ
- Форматы: JPG, PNG, GIF
- Соотношение сторон: 16:9 (рекомендуемое)
- Валидация размера при загрузке
- Отображение без обрезания (object-fit: contain)

## Безопасность

- CSRF-защита на всех формах
- Валидация файлов на стороне сервера
- Встроенная защита Django от XSS-атак
- Разграничение доступа по ролям

## Требования к контенту

- Контент должен быть оригинальным и полезным для аудитории
- Рекомендуется избегать одежды в мелкую клетку/полоску (эффект муара)
- Рекомендуется смотреть в камеру и использовать открытые позы
- Видео должно иметь чёткую структуру
